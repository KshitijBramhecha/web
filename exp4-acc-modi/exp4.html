Output:

Explanation of Access Modifiers

  • public - Can be accessed from anywhere (default if not specified)
  • private - Can only be accessed within the class
  • protected - Can be accessed within the class and its subclasses

In this example:

  • The Person class has public, private, and protected properties
  • The Employee class extends Person and can access public and protected properties, but not private ones
